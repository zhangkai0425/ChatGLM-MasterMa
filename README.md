接下来，您可以使用以下代码来编写一个简单的希尔排序程序：

```python
import numpy as np

def shell_sort(arr):
    n = len(arr)
    # 遍历所有数组元素
    for i in range(n):
        # 最后 i 个元素已经排好序
        for j in range(0, n-i-1):
            # 如果当前元素比下一个元素大，则交换它们
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]

# 要排序的数组
my_array = np.array([5, 2, 9, 1, 5, 6])

# 调用希尔排序算法
shell_sort(my_array)

# 打印排序后的数组
print("排序后的数组：")
print(my_array)
```

以上代码将打印出以下结果：

```
排序后的数组：
[1, 2, 5, 5, 6, 9]
```
